Path Finding Algorithm Visualization

Visualize how a path finding algorithm works using Brython. In this page, I'm going to implement 2 different algorithms to find the shortest path between two nodes: Breadth First Search (BFS) and A-star. Here are quick words on how they work:

  • Breadth First Search (BFS) algorithm searches for the ending point by layers, where each layer is the next direct unsearched neighbors of the nodes in the previous layer. This algorithm ensures that once the ending point is found, the path being traced back is the shortest possible path. More information on this Wikipedia page.
  • A-Star (A*) algorithm also provides the shortest path solution, and even better, a faster execution time than BFS. Instead of searching (nearly) exhaustively, the algorithm calculates at each round an F-score of each node (based on a good estimation of the distance with the ending point) then decide where to search next based on the calculation. More information on this Wikipedia page.
